An enterprise-style Prisoner Management System built using **PHP and MySQL**, designed to digitize prison administration with structured workflows, role-based dashboards, and secure access control.

---

## 📌 System Overview

This platform centralizes prison operations by managing inmates, officers, lawyers, administrators, and visitors through dedicated role-based modules.  

The system ensures organized institutional governance using secure routing, relational database design, and controlled access mechanisms.

---

## 🏗 Core Modules

### 🔹 Admin Module
- Manage officers, jailors, lawyers, and prison sections
- Monitor prison statistics and operational data
- System-wide control and administrative oversight

### 🔹 Officer Module
- Add and manage prisoner records
- Assign crimes and update inmate status
- Handle visit request approvals
- Maintain prisoner risk-level classification

### 🔹 Lawyer Module
- Access assigned prisoner case details
- Manage legal notes and case intelligence
- Submit visit requests for inmates
- Track approval status workflow

### 🔹 Prisoner Module
- Role-based login access
- Access rehabilitation content
- Integrated **YouTube motivational & educational videos**
- Structured digital engagement for inmate development

### 🔹 Visitor & Visit Approval Workflow
- Visitor registration system
- Scheduled visit requests
- Multi-level approval process (Officer & Admin)
- Organized visit tracking and status updates

---

## 🔐 Security & Architecture

- Centralized routing via `public/index.php`
- Role-based authentication system
- Session-controlled access management
- Structured MVC-like modular architecture
- Normalized relational database schema
- Secure workflow handling
